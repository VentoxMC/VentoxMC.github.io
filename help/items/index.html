Utility Items

Wrenches

Wrenches are a useful utility-item players can purchase or find. There are multiple different types of wrenches.

Item Wrench

The Item Wrench is a utility wrench which can be used to retrieve and pickup Spawners as well as destroy [Team Cores]Spawners.

This wrench has an limited number of uses and takes a few seconds to use, per block. The number of uses and duration of use changes depending on where the Item Wrench is obtained from.

The Item Wrench can be purchased at Spawn for a range of money options, but can also be obtained through events.

Transport Wrench

The Transport Wrench is a team-based wrench which can be used to retrieve and pickup Spawners within Team Territory.

This wrench has an infinite number of uses and takes 10-seconds to use, per Spawner. The player who uses this wrench must have the proper Team Permission in order for it to actually work.

The Transport Wrench can be purchased at Spawn for $75,000.

Ice Pick

The Ice Pick is a Christmas season exclusive which allows players to fish at the Snowdrift Shores.
